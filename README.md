# Patients EHR SQL Analysis Project

## Objective
Analyze healthcare patient records using PostgreSQL.

## Tools
- PostgreSQL 18
- pgAdmin 4
- SQL

## Workflow

1. Create raw table
2. Import CSV dataset
3. Create staging table
4. Clean data:
   - Remove duplicates
   - Standardize gender
   - Convert date fields
   - Handle missing values
5. Perform analysis

## SQL Skills Demonstrated

- CREATE TABLE
- COPY / data loading
- CTEs
- Window functions
- ROW_NUMBER()
- Data cleaning
- Aggregations
- Joins
- Date functions
